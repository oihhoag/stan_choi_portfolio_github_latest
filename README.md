# Stan Choi Portfolio

GitHub-ready static portfolio package.

## Structure

```text
index.html
assets/
  css/
    styles.css
  js/
    main.js
  images/
```

## Publish with GitHub Pages

1. Upload everything inside this folder to the root of your GitHub repository.
2. Open **Settings → Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)`.
5. Save.
